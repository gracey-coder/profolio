Introduction

Doctypes

<!Doctype>

  1. Should declare before <html> tag
  2. Case insensitive

Heading

  1. Provides 6 types header tag
  2. Search engines and other user agents usually index page content based on heading elements
  3. Correct structure for headings is important

Paragraphs

<p> will ignore extra space or extra line

Tags Description
<p> Defines a paragraphs
<br> New line
<pre> Defines pre-formatted text
Text Formatting

Highlighting

  1. Use <mark>
  2. It will highlight the text in yellow, it can change in CSS
  3. Example:
    Here is some content from an article that contains the <mark> searched query </mark>.

Bold, Italic, Underline

  1. Bold Text: <strong> or <b> tag
    - <strong> indicates that the text is fundamentally or semantically important to the surrounding text.
    - <b> ˗indicates no such importance and simply represents text that should be bold.
  2. Italicize text: <em> or <i> tag
    - <em> indicate that the text should have extra emphasize or need to be stressed.
    - <i> simply represents text which should be set off from the normal text around it.
  3. Underline: <u>

Abbreviation

Use <abr>, here's example: (mouse hover the word of HTML)
I like to write <abr title="Hypertext Markup Language">HTML</abr>!

Inserted, Deleted, Stricken

Mark text as inserted: <ins> tag
Mark text as deleted: <del> tag
Strike through text: <s> tag
Example:
<ins>inserted text</ins> <del>deleted text</del> <s>struck through text</s>

Superscript and Subscript

Offset text upward: <sup>
Offset text downward: <sub>
<sup>superscript</sup>

Main element

  1. The content inside the <main> element in a document should be unique
  2. It should not contain any content that is repeated across documents such as sidebars, navigation links, copyright information, site logos, and search form
Here's example:
<main> <h1>Most Popular Browsers </h1> <p>Chrome, Firefox, and Edge are the most used browsers today.</p> <article> <h2>Google Chrome </h2> <p>Google Chrome is a web browser developed by Google, released in 2008. Chrome is the world's most popular web browser today!</p> </article> <article> <h2>Mozilla Firefox</h2> <p>Mozilla Firefox is an open-source web browser developed by Mozilla. Firefox has been the second most popular web browser since January, 2018.</p> </article> <article> <h2>Microsoft Edge</h2> <p>Microsoft Edge is a web browser developed by Microsoft, released in 2015. Microsoft Edge replaced Internet Explorer.</p> </article> </main>

Make an element scrollable

In CSS. use <overflow-x>, <overflow-y> or <overflow> to define it
Here's example:
#div{ overflow-y: auto; overflow-x: hidden; }

CSS - Viewport Height(vh)

This unit is based on the height of viewport.
A value of 1vh = 1% of the viewport height

CSS - Viewport Width(vw)

This unit is based on the width of viewport.
A value of 1vw = 1% of the viewport width

Display Properties - Grid

After define code below in CSS display: grid; Here's the grid properties can be used:

  1. grid-template-columns: define how many columns inside a gride,
    You can also define each column width
    See example 1
  2. grid-template-rows: define how many rows inside a gride,
    You can also define each column width
    See example 2
  3. You can use value repeat() for both grid-template-columns and grid-template-rows
    See example 3
  4. When using repeat() value:
    - For column/row number, instead of direct define with number , you can use auto-fit or auto-fill
    - For column width / row height, instead of direct define with number, you can use minmax(min no, max no)
    You may use this to create responsive website.
    See example 4
  5. grid-column-gap / grid-row-gap: set the gap between columns/rows
    or you can use grid-gap: column-width row-width
    See example 5
  6. You can use grid-template-area to define columns and rows like a table in upper level div
    In each child div, remember define grid-area
    See example 6

Example 1: Define 3 columns, each width is 60px <style type="text/css"> .grid{ display:grid; grid-template-columns: 60px 60px 60px; } .tile{ height: 30px; color: #5CDB95; background-color: #05386B; } </script> <div class="grid"> <div class="tile">1</div> <div class="tile">2</div> <div class="tile">3</div> <div class="tile">4</div> <div class="tile">5</div> </div>
